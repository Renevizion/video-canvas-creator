## What You Can Do

### 1. Preview
- Click any composition in sidebar
- Scrub through timeline
- Play/pause animation
- Step frame by frame

### 2. Customize
- Edit composition props
- Change colors
- Modify text
- Adjust timing

### 3. Render
- Click "Render" button
- Choose codec (H.264, H.265, VP8, VP9)
- Select quality
- Export to MP4

### 4. Develop
- Hot reload on file changes
- See TypeScript errors
- Debug in browser console
